# YOLO-Based Supermarket Shelf Product Detection and Out-of-Stock Analysis

Computer Vision final project focused on:
- detecting supermarket shelf products using YOLO
- counting visible product facings
- identifying low-stock and out-of-stock categories

## Planned Classes
- water
- milk
- juice
- cereal
- chips
- pasta

## Main Workflow
1. Prepare and inspect the dataset
2. Fine-tune YOLO on selected shelf classes
3. Run inference on test images
4. Count detections per class
5. Apply stock-status rules
6. Evaluate results using detection and stock metrics
